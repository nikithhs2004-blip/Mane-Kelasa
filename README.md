# ಮನೆ ಕೆಲಸ — Mane Kelasa

A Hyper-Local Work Directory (Digital Naka) for domestic workers in Bengaluru.

## Features
- Worker & Seeker role-based login
- Workers toggle availability in real-time
- Seekers search workers by skill and town
- One-tap call
- Post job requests
- Thumbs up rating system
- LocationIQ map integration
- Firebase Auth + Firestore backend

## Tech Stack
- Android (Kotlin)
- Firebase Authentication
- Cloud Firestore
- LocationIQ Maps
- Material Design

## Setup
1. Clone the repo
2. Add `google-services.json` from Firebase Console into `app/`
3. Add LocationIQ API key in `app/build.gradle.kts`
4. Run in Android Studio
